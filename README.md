# 5M06-Operation---Game

#                         Welcome to the 5M06 Operation game
Be part of this operation to save the system from the black hat hackers that are trying to spread their pinkish viruses in the system and ruin the virtual world. Through collecting as much as possible of bitcoins and keys you would save the system.

#                                   How to play:
- Use the arrow keys [Right, Left, Up] to move and jump. click [Up] many times to FLY!
- Make sure to click the small box (with headphones icon) to hear the soundeffects
